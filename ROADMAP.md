# Roadmap

## Phase 1: Local Blockchain
- Block and chain data structures
- SHA-256 hashing and chain validation
- Basic transaction model

## Phase 2: CPU Mining
- Proof-of-work with adjustable difficulty
- Block reward and coinbase transaction
- Mining loop (CPU only)

## Phase 3: Wallet
- Key pair generation (ECDSA)
- Transaction signing and verification
- Address generation and balance tracking

## Phase 4: Networking
- Peer-to-peer node connections
- Block and transaction propagation
- Chain sync and conflict resolution (longest chain)

## Phase 5: Inscriptions
- Embed arbitrary data in transactions
- Indexer to read and serve inscriptions
- Basic inscription standard

## Phase 6: Privacy Research
- Investigate confidential transactions
- Evaluate stealth addresses or ring signatures
- Prototype and document findings
